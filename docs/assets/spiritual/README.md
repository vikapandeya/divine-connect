Generated spiritual website imagery lives in this folder.

Run `npm run images:generate` to create:

- `mandir.jpg`
- `prasad.jpg`
- `aarti.jpg`
- `meditation.jpg`
- `pooja.jpg`

The generator uses `OPENAI_API_KEY` when available and falls back to free thematic image downloads when it is missing.
The fallback path is meant to keep the app visually relevant even without a paid image API key.
